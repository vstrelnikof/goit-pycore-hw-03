## Персональний помічник Python Programming

Встановлення:
```
git clone https://github.com/vstrelnikof/edu-prj-01.git
cd project
python3 main.py
```

Використання:
- add_contact – додати контакт
- list_contacts – показати всі контакти
- search_contact – пошук контакту
- edit_contact – редагувати контакт
- delete_contact – видалити контакт
- birthdays – показати дні народження
- add_note – додати нотатку з тегами
- list_notes – показати всі нотатки
- search_note – пошук нотатки (по тексту або тегу)
- edit_note – редагувати нотатку
- delete_note – видалити нотатку
- exit – вихід з програми
